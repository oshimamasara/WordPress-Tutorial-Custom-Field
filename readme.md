## テスト方法

1. git clone https://github.com/oshimamasara/WordPress-Tutorial-Custom-Field.git
2. WordPress スタート（XAMPP or Docker）
3. phpMyAdmin 起動
4. DBフォルダ内のデータをデータをインポート
5. WordPress管理画面　→　投稿画面からカスタムフィールドを確認
